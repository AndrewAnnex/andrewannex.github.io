Using SpiceyPy to map the Curiosity rover (MSL)

Apr 15, 2020 9 min read

Using SpiceyPy to map the Curiosity rover (MSL).

note:

Here is a link to a gist jupter notebook of this tutorial

In this post, I provide a worked out tutorial for using my library SpiceyPy and some of the standard python libraries to make a map of the Curiosity Rover (Mars Science Laboratory) position on the surface of Mars for a number of years.

Step 1: getting kernels

I am interested in MSL's movements from landing in 2012 to 2016, so I used this PDS tool to download the kernel list I need from 2012-08-07 to 2016-08-06 at https://naif.jpl.nasa.gov/cgi-bin/subsetds.pl?dataset=msl-m-spice-6-v1.0/mslsp_1000 . I know MSL landed on the 6th but I don't think it went very far in the first day.

This list is also probably way more comprehensive then what we need just to plot the rover on a map, but that is fine!

spiceypy.ktotal('ALL')

Step 4: Use SpiceyPy to get the longitude and latitude of the rover through time

we will pick a time range from August 8th 2012 to August 1st 2016, which is around 1450 days. I will use linspace to enumerate 4350 times between these times so that I will definitely have at least 1 position per sol. Remember, Mars sols are slightly longer than 24 hours. Finer time steps are likely possible but that would require a huge query or more targeted queries surrounding known traverses.

sol0 = spiceypy.str2et("2012 AUG 08 00:00:00.000 UTC")
sol1450 = spiceypy.str2et("2016 AUG 01 00:00:00.000 UTC")
times = np.linspace(sol0, sol1450, num=1450*3, endpoint=True)
len(times)

4350

great, lets grab the positions using spkpos in SpiceyPy

positions, lttimes = spiceypy.spkpos("MARS SCIENCE LABORATORY", times, "IAU_MARS", "lt+s", "MARS")
len(positions)

4350

now we need to convert it to lon lat, let's go with planetocentric

r, lon, lat = np.array([spiceypy.reclat(p) for p in positions]).T
lon *= spiceypy.dpr() # you could also just use numpy.degrees for this...
lat *= spiceypy.dpr()

Great, so let's make a simple map to show the traverse. Here I plot the line of points and the marker points for each position, with a colormap corresponding to increasingly yellow through time.

plt.figure(figsize=(6,6))
plt.plot(lon,lat)
plt.scatter(lon,lat, c=times)
plt.title('MSL from 2012-08-08 to 2016-08-01')

Step 5: make a very pretty map

Let's make an interactive map with folium. I will be using the USGS Astrogeology center's map layers hosted by ESRI for basemaps. I've included the shaded MOLA+HRSC topography layer, the global CTX mosaic layer, and the uncontrolled (more on this in a bit) HiRISE mosaic layer. Note that there is a significant (hundred+ meters) miss-alignment between where the MSL positions are plotted in longitude and latitude from the HiRISE basemap. This is due to the basemap of hirise being produced as a "uncontrolled" mosaic so the placement of the images is not exact. I expect that the position of the rover in longitude and latitude is more correct as the spice kernels I used are from the PDS archive.

import folium

tiles='https://astro.arcgis.com/arcgis/rest/services/OnMars/MColorDEM/MapServer/tile/{z}/{y}/{x}'

m = folium.Map(location=[-4.7, 137.35],
               zoom_start=11,
               max_zoom=18,
               tiles=None, 
               crs='EPSG4326')

folium.TileLayer(
    max_native_zoom=7,
    tiles=tiles,
    attr='usgs/esri',
    crs='EPSG4326'
).add_to(m)

ctx = folium.raster_layers.TileLayer(
    tiles='https://astro.arcgis.com/arcgis/rest/services/OnMars/CTX/MapServer/tile/{z}/{y}/{x}',
    max_native_zoom=12,
    attr='usgs/caltech',
    name='ctx', 
    opacity=1,
    overlay=True, 
    control=True,
    crs='EPSG4326'
)
ctx.add_to(m)

hirise = folium.raster_layers.TileLayer(
    tiles='https://astro.arcgis.com/arcgis/rest/services/OnMars/HiRISE/MapServer/tile/{z}/{y}/{x}',
    max_native_zoom=17,
    attr='usgs/esri',
    name='hirise', 
    opacity=1,
    overlay=True, 
    control=True,
    crs='EPSG4326'
)
hirise.add_to(m)

# add points from positions
utc_times = spiceypy.et2utc(times, 'C', 0)
folium.PolyLine(list(zip(lat, lon)), color='green').add_to(m)
for i, (lo, la) in enumerate(list(zip(lon,lat))[::10]):
    folium.Marker((la, lo), tooltip=f'{utc_times[i]}').add_to(m)
folium.LayerControl().add_to(m)
